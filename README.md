# ax-npu-kit-650
 
## 简介

**ax-npu-kit-650** 由开源社区友情开发。本项目的为向社区开发者提供 AX650 SDK 中预装智能算法的**SDK 参考设计**，用户可根据自己的需求拓展 **Pipeline** 应用。

**已支持芯片**
- AX650N

**已支持开发板**
- AXera-Pi Pro(AX650N)
