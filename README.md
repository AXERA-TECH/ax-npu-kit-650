# ax-skel-650
 
## 简介

**ax-skel-650** 由 **[爱芯元智](https://www.axera-tech.com/)** 主导开发。本项目为爱芯元智自研的算法Pipeline库，用户可根据自己的需求拓展Pipeline应用。

**已支持芯片**
- AX650N

**已支持开发板**
- AXera-Pi Pro(AX650N)
